 ## How to set up
 1. Download zip and extract to vscode
 2. npm i (this install all packages)
 3. create .env file with   JWT_SECRET = YOUR_SECRET
 4. npm start
 5. app will be available in http://localhost:3000/
 ## Routes
 1. GET (/api/v1/dashboard)
 2. POST (/api/v1/login)
